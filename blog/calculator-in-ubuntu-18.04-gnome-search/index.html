Calculator in Ubuntu 18.04 GNOME search

本文介紹如何在 Ubuntu 18.04 dash 直接使用計算機預覽

January 12, 2019 Jerry Wang

1 minute read

  • Twitter

  • Facebook

前言

看到 gslin 大大的直接在-ubuntu-的-unity-上直接計算，後來想到 Ubuntu 18.04 應該也要有這功能才對，於是就找了一下。

過程

在 18.04 desktop 預設是用 GNOME 作為 GUI 介面，並使用了 Snap 安裝預設套件， 包含了 GNOME 及 計算機 等相關套件。

但是透過 snap 安裝的一些套件，缺少了一些功能。

// 移除透過 snap 安裝 calculator 並重新透過 apt 重新安裝
sudo snap remove gnome-calculator
sudo apt install gnome-calculator

可以在 dash 計算了～
